Ôn lại về DOM

  • Document Object Model - Mô hình hoá tài liệu dưới dạng đối tượng.
  • Bao gồm 3 thành phần chính (Nodes): Element Node, Attribute Node, Text Node
  • Javascript có thể tương tác với HTML thông qua DOM (đọc, thêm, xoá, sửa..)
  • Có 5 cách để query Element trong DOM:
    getElementById,
    getElementsByClassName,
    getElementsByTagName,
    querySelector,
    querySelectorAll
